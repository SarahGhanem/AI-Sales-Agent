Overview
This workflow implements an intelligent chatbot for a training center website. It automatically answers questions about courses, prices, and schedules, 
captures prospective information, and routes conversations  with built-in guardrails.
The system is designed to operate 24/7 without human intervention for standard inquiries.

Prerequisites

n8n instance (self-hosted or cloud)
OpenAI API key
Google Cloud project with Gemini API enabled
Gmail account for sending emails to clients and the sales team.
